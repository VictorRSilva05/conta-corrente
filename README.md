# ContaCorrente

![](https://i.imgur.com/BFaDIy3.gif)

## Descrição

Esse projeto é uma aplicação de conta corrente, onde o usuário pode criar uma conta, depositar, sacar e transferir valores entre contas. O projeto foi desenvolvido com a interface gráfica do Windows Forms.

## Tecnologias utilizadas

![](https://img.shields.io/badge/.NET-6.0-blue)
![](https://img.shields.io/badge/C%23-9.0-blue)
![](https://img.shields.io/badge/Windows%20Forms-5.0-blue)

## Funcionamento

O usuário informa as informações iniciais necessárias, que são
utilizadas para construir o objeto da conta e armazena-las na lista
de contas. Após isso, o usuário pode realizar operações de depósito, saque e transferência entre contas. O saldo da conta é atualizado automaticamente após cada operação.


**Clone o repositório**

```
https://github.com/VictorRSilva05/conta-corrente.git
```

**Navegue até a pasta raiz da solução**

```
cd jogo-da-forca-2025
```

**Restaure as dependências**

```
dotnet restore
```

**Navegue até a pasta do projeto**

```
cd JogoDaForca.ConsoleApp
```

**Execute o projeto**

```
dotnet run
```
